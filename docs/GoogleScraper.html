Google Scraper

googleScraper

Returns Google SERPs URLs for a given keyword. Does not return universal search

googleScraper

Function

=googleScraper(keyword, optTld, optStart)

Parameters

Name Type Description
keyword string Keyword
optTld string (Optional) Top level domain (eg: ".co.uk". Defaults to ".com")
optStart string (Optional) Sets the starting offset for results (defaults to 0)
