Raven API

getRavenAllRank

Gets the rankings for all your keywords for one domain on a specific date.

getRavenCountKeywordsOnPage

Gets the number of tracked keywords on a specific page

getRavenDomains

Gets a list of all the domains in your account

getRavenKeywords

Gets a list of all the keywords you're currently tracking

getRavenRank

Gets one metric (such as ranking) for a single keyword from a specific engine, on a specific day.

getRavenAllRank

Function

=getRavenAllRank(domainName, date)

Details

The rankings be returned in the following columns:

  • keyword
  • engine
  • rank
  • status
  • date
  • url

    • Parameters

    Name Type Description
    domainName string The raven domain name
    date date The date you want rankings from

    Returns

    Column Description
    Returns a 2D array of rankings

    Examples

     

    Cells:
   A1: seerinteractive.com
   A2: 11/01/2011
=getRavenAllRank(A1, A2)
// Returns the Google ranking for the keyword "seo company" on 11/01/11

    getRavenCountKeywordsOnPage

    Function

    =getRavenCountKeywordsOnPage(domainName, date, engine, page)

    Details

    Gets the number of tracked keywords on single page of a search engine.

    Parameters

    Name Type Description
    domainName string The raven domain name
    date date The date you want rankings from
    engine string Search engine
    page integer Page number you want to get a count from

    Returns

    Column Description
    The number of keywords that appears on a specific page

    Examples

     

    Cells:
   A1: seerinteractive.com
   A2: 11/01/2011
   A3: google
   A4: 1 
=getRavenCountKeywordsOnPage(A1, A2, A3, A4)
// Gets the number of keywords on page 1 on 11/01/2011
