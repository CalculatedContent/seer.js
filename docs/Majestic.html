Majestic API

getMajesticBackLinks

Gets a list of backlinks from Majestic SEO

getMajesticindexItemInfo

Retuns index information about a URL

getMajesticindexItemInfo

Returns backlinks for a URL

getMajesticMaxBacklinksShown

Shortcut that gets the max number of backlinks this API key can show

getMajesticSubdomainData

Get information on referring subdomain's backlinks from Majestic SEO. For example, if a backlink was "http://www.domain.com/blog.html" then the information you will get is just for "www.domain.com"

getMajesticSubscriptionInfo

Returns information on existing subscriptions with current resource usage values provided.

getMajesticBackLinks

Function

=getMajesticBackLinks(url, optSource, optLimit)

Details

Gets a list of backlinks from Majestic SEO. The maximum number of URLs it pulls is set in the Settings sheet.

Parameters

Name Type Description
url string URL to get backlinks for
optSource string (optional) Data source to get information from (can be "fresh" or "historic") Default is "fresh"
optLimit integer (optional) Override the maximum number of URLs to pull.

Returns

Column Description
Domain Root domain of the backlink
AlexaRank Alexa Rank of the root domain (only valid for top 1 mln domains), see more for details {@link http
RefDomains Number of referring domains pointing to that root domain
ExtBackLinks Number of external backlinks pointing to the root domain
IndexedURLs Number of unique URLs in index (includes crawled and uncrawled)
CrawledURLs Number of unique URLs that were crawled
FirstCrawled Date when domain was first crawled
LastSuccessfulCrawl Date when domain was last successfully crawled
IP IPv4 Address of the domain (note for hosts with multiple IPs only one IP will be taken)
SubNet Calculated from IP
CountryCode Geo-located country of hosting based on IP address
TLD Top level domain of the root domain
DomainID The domainID numbers will correspond with the ID numbers found from =getMajesticDomainInfo

Examples

Get backlinks for a URL:
Return the backlinks from Majestic for www.seerinteractive.com/blog 

=getMajesticBacklinks("www.seerinteractive.com/blog")

getMajesticindexItemInfo

Function

=getMajesticindexItemInfo(urls)

Details

This function returns key information about "index items." If you give it a root domain name (eg: "seerinteractive.com") then you will get index information for all URLs belonging to that root domain.

If you give it a subdomain (eg: "www.seerinteractive.com") then you will get index information that belong to just that subdomain.

If you give it an exact URL (eg: "http://www.seerinteractive.com/blog") then you will get index information for only that URL.

Parameters

Name Type Description
urls array or string URL as an array

Examples

Get Majestic index items:
Return majestic index items for a range of URLs 

Cells:
   A1: www.seerinteractive.com
   A2: www.nasty-competitor.com/blog
   A3: www.another-guy.com
=getMajesticIndexItemInfo(A1:C3)

getMajesticindexItemInfo

Function

=getMajesticindexItemInfo(url, datasource, maxSourceUrls, showDomainInfo, getUrlData, getSubDomainData, getRootDomain, maxSourceURLsPerRefDomain)

Details

This will return backlinks (up to a limit of the subscription level) for any URL, domain or subdomain.

Parameters

Name Type Description
url string See: http://bit.ly/tb91ri
datasource string See: http://bit.ly/tb91ri
maxSourceUrls int Defaults to the hard limit setting or the max backlinks your account will show whichever is lower. Use 0 to force the maximum backlinks.
showDomainInfo boolean See: http://bit.ly/tb91ri
getUrlData boolean See: http://bit.ly/tb91ri
getSubDomainData boolean See: http://bit.ly/tb91ri
getRootDomain boolean See: http://bit.ly/tb91ri
maxSourceURLsPerRefDomain int See: http://bit.ly/tb91ri

Examples

Get top backlinks for a URL:
Get the top results up to the BACKLINK_HARD_LIMIT setting 

=getMajesticTopBackLinks("http://www.seerinteractive.com")

Get the top 100 results:
Get the top 100 backlink results 

=getMajesticTopBackLinks("http://www.seerinteractive.com", 100)

Get all the backlinks for a URL:
Get the the maximum resuts. !!WARNING!! Most likely will make Google Docs hang!! 

=getMajesticTopBackLinks("http://www.seerinteractive.com", 0)
