Redirection Tools

createHtaccessRedirect

Generates .htaccess code to 301 redirect an old link to a new one.

createHtaccessRedirect

Function

=createHtaccessRedirect(oldUrl, newUrl)

Details

Inspired by http://www.thegooglecache.com/rants-and-raves/new-pagerank-recovery-tool/

Parameters

Name Type Description
oldUrl string The old URL you want to 301 redirect
newUrl string The new URL

Examples

 

A1: http://www.domain.com/old-page
B1: http://www.domain.com/new-page
C1: =createHtaccessRedirect(A1, B1)
